Rotina de treino 01 (RT01)

    Os treinos abaixam são uma divisão de 05 dias de treino e 02 de descanso, possibilitando a treino do corpo completo durante a semana e repouso total aos fins de semana.

  1. Treino de peito

    • Supino Reto 3x10
    • Supino Inclinado 3x10
    • Cruxifixo na polia 3x10
    • Supino Declinado 3x10

    Realizar intervalo de 1:30m entre cada serie

    O periodo de descanso pode ser adaptado para o nivel de cansaço fisico

  2. Treino 01 de perna

    • Agachamento 3x10
    • Cadeira extensora 3x10
    • Leg 45° 3x10
    • Agachamento bulgaro 3x10

    Realizar intervalo de 1:30m entre cada serie

    O periodo de descanso pode ser adaptado para o nivel de cansaço fisico

  3. Treino de costas

    • Barra fixa 3x10
    • Puxada com triangulo 3x10
    • Remada unilateral 3x10
    • Remada aberta 3x10

    Realizar intervalo de 1:30m entre cada serie

    O periodo de descanso pode ser adaptado para o nivel de cansaço fisico

  4. Treino 02 de pernas

    • Cadeira flexora 3x10
    • Mesa flexora unilateral 3x10
    • Stiff 3x10

    Realizar intervalo de 1:30m entre cada serie

    O periodo de descanso pode ser adaptado para o nivel de cansaço fisico

  5. Treino de ombro

    • Desenvolvimento com alteres 3x10
    • Elevação frontal 3x10
    • Elevação Lateral 3x10
    • Cruxifixo invertido 3x10

    Realizar intervalo de 1:30m entre cada serie

    O periodo de descanso pode ser adaptado para o nivel de cansaço fisico

Lembre-se de descansar o corpo aos fins de semana
